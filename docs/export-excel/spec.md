# Excel 匯出功能需求

## 功能說明

從 BorderCollie 網頁版匯出靜態 Excel 檔案 (`.xlsx`)，包含：

1. **ProjectGantt 工作表** - 專案甘特圖
2. **PersonGantt 工作表** - 人力甘特圖

## 需求細節

### 基本需求

- 直接產出靜態圖表，不需要互動編輯功能
- 使用 cell 背景色繪製甘特條塊
- 包含年/月時間軸標題列
- 支援 Excel 2016+ 及 LibreOffice/Google Sheets

### ProjectGantt 工作表

- 縱軸為專案名稱
- 橫軸為時間 (年/月)
- 各階段以彩色儲存格顯示
- 專案名稱跨列合併
- Pending 專案不顯示

### PersonGantt 工作表

- 縱軸為人員名稱
- 橫軸為時間 (年/月)
- 每位人員第一行顯示月負載數值
- 負載 > 1.1 顯示紅色警示
- 負載 ≤ 1.1 顯示綠色
- 下方列出該人員參與的專案階段

### 樣式需求

- 年份標題: 深色背景、白色粗體文字
- 月份標題: 淺色背景、置中
- 專案/人員名稱: 粗體、凍結欄
- 甘特條塊: 與網頁版相同的專案顏色
